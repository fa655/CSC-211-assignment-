Basic Tag Usage

Tags are keywords that provide instructions about the web page.

Lists of Html Tags and their functions

1.div tag

Functions: Defines a paragraph of text.

3. h1 to h6 tags

Functions: Defines headings of different levels.

4.img tag

Functions: Embeds an image into the document.

5. a tag

Functions: Creates a hyperlink to another page or resource.

6. ul and ol tags

Functions: Define unordered (bulleted) and ordered (numbered) lists respectively.

7. li tag

Functions: Defines anitem within a list

8.table,tr,td, and th tags

Functions: Define tabular data table creates a row,td creates a data cell, and th creates a header cell.

9. form tag

Functions: Creates a form for user input.

10. input tag

Functions: Defines various types ofinput fields within a form, such as text fields,buttons, and checkboxes.
